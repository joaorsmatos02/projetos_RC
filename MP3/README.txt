O ficheiro TrafficAnalysis.java contém o código java utilizado para analisar os traces.
Na elaboração deste código foi considerada a possibilidade de um trace conter mais do que um ataque,
sendo que os ficheiros com a informação para a geração de gráficos não são criados criados diretamente
com o nome "Q7_Grupo14.txt" de forma a evitar conflitos caso seja necessario mais do que um ficheiro.
Em vez disso, são criados com o nome "results_XXX.txt" em que XXX representa o tipo de ataque (POD, SYN,
UDP). Para efeitos de entrega, o nome do ficheiro obtido foi alterado para estar em conformidade com o 
enunciado.

O ficheiro Q7_Grupo14.r contem o código R usado para gerar o gráfico. 
Novamente considerou-se a possibilidade de ter que gerar mais que um gráfico, portanto este código 
utiliza os nomes dos ficheiros tal como gerados no código java. Para utilizar vai ser necessário 
alterar os caminhos dos ficheiros.
